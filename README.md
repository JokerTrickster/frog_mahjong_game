# 개굴작 보드게임

### 프로젝트 설명
참새작 보드게임을 모티브로 웹에서 여러 유저들과 플레이

### 기능
1. 로그인 기능
2. 방 개설 (최대 1개 추후 여유가 되면 여러개 개설되도록 확장 고려)
3. 게임 준비 및 시작 기능
4. 패 섞기
5. 패 버리기
6. 패 가져오기
7. 상대방 버린패 가져오기
8. 점수 계산
9. 점수 계산 설명서 보기
