### MVP 기능
1. 인증 기능  
1-1. 회원가입 및 로그인 (구글, 이메일)  
1-2. 비밀번호 찾기 (이메일 인증)  
2. 매칭 기능  
3. 함께하기 기능 & 방 생성(인증코드 발급)  
4. 게임 플레이  
 4-1. 패 가져오기  
 4-2. 패 버리기  
 4-3. 도라 선택하기  
 4-4. 쯔모 외치기  
 4-5. 론 외치기  
5. 신고하기 기능 (이메일 및 대시보드 연동)
